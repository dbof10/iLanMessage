# Project purpose

This project exists to build a system for **sharing data between machines**: files, images, video, code, and other payloads, sent **from one machine to another** where the receiving side is an **MDM-managed device**.

## Product goal

Ship a **high-quality web experience** that makes it **easy** to move **files, structured data, and plain text** between those endpoints—with clear flows, reliable transfers, and UX that feels intentional rather than bolted on.

When implementing features, prefer choices that improve discoverability, feedback during transfer, error handling, and overall usability for this cross-machine, MDM-context workflow.
